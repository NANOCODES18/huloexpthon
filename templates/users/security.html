Google Authentication

Google Authentication is a dynamic password tool that works like SMS dynamic verification

A dynamic verification code is generated every 30s after binding and the verification code can be used for security verification of operations such as login,coin withdrawal and modifications of security settings.

  1. IOS users log in to the App Store to search for 'Authenticator'.
  2. Android users log in to the app store or use the mobile browser to search for 'Google Authenticator'.
  3. Open Google Authentication, scan the QR code below or manually enter the following key to add a verification token.
  4. The key is used to retrieve the Google Authenticator when the phone is replaced or lost. Be sure to save the following key backup before binding
